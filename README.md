Skiltr is an open-source Guild Wars fan project focused on advanced skill exploration, build creation, and structured analysis.



⚠️ \*\*Status:\*\* Beta – actively under development. ⚠️

Expect bugs, incomplete features, and ongoing refactoring.



\## Live Preview



You can view the current development version at:



https://skiltr.magical.ch



\## Purpose



Skiltr aims to provide:



\- Overengineered skill search and filtering

\- Advanced build creation and editing

\- Personal categorisation and study of skills

\- Persistent builds and containers

\- Key-based identity system using cryptography (no traditional accounts)



The long-term vision is a powerful research and build management tool for serious Guild Wars players.



\## Setup



1\. Copy `inc/db.example.php` to `inc/db.php`

2\. Configure database credentials

3\. Import the database schema

4\. Serve via PHP (Apache/Nginx)
